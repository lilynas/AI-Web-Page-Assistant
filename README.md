# AI 网页智能总结助手 (AI Web Page Assistant)

这是一个强大且高度可定制的 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 脚本，专为在网页阅读时提供即时的 AI 协助而设计。它能够优雅地提取网页正文内容，让您在不离开阅读环境的情况下进行对话、总结和深度追问。

## 主要功能特点

- **沉浸式阅读总结**：
  - 内置 Mozilla `Readability.js`，智能剔除网页中的广告、导航栏和侧边栏等干扰信息，精准提取纯净的正文文本。
  - 大幅节约 AI 对话大模型的 Context Token 使用量，不仅省钱，还能让 AI 的总结更精准。

- **超强交互界面 (UI)**：
  - **拖拽悬浮球**：右下角的蓝色悬浮机器人按钮支持全屏自由拖拽。
  - **非侵入式自动隐藏**：如果不将鼠标悬停在上方，悬浮球会在 3 秒后变为半透明并自动缩进屏幕边缘，绝不遮挡您的阅读视线。
  - **响应式侧边栏**：点击悬浮球（或使用快捷键 `Ctrl+J` / `Cmd+J`）即可呼出聊天面板。桌面端优雅地停靠在右侧，移动设备端则自底部流畅滑出。

- **全局极客定制化**：
  - 原生调用 Tampermonkey 的 `GM_xmlhttpRequest`，**完美绕过浏览器 CORS 跨域限制**。
  - 支持任意兼容 OpenAI 格式的接口代理（例如：DeepSeek, 第三方 API 聚合站等），填入您的 `API Base URL` 和 `API Key` 即可直连。
  - 一键支持**流式输出 (打字机效果)**，聊天响应丝滑顺畅。

- **多面手提示词管家**：
  - 出厂预设 **6 大强力助手**模板（包含：📝 通用总结、💻 技术文档、📰 新闻文章、📚 教程指南、🔬 学术论文、🎁 产品介绍）。
  - 在设置面板中支持动态无缝创建、编辑和删除您专属的自定义提示词（Prompt）。

- **智能追问引擎**：每一次 AI 回答结束后，底部的追问模块会自动生成 3 个高质量的相关建议问题，点击即可顺滑追问，深入探索文章核心。

## 📥 安装指南

1. 确保您的浏览器已安装并启用 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. 点击此处 [直接安装脚本](https://raw.githubusercontent.com/lilynas/AI-Web-Page-Assistant/main/ai_web_assistant.user.js)，或者复制本仓库内 `ai_web_assistant.user.js` 的源码并在 Tampermonkey 面板中新建脚本粘贴保存。

## 🚀 快速起步

1. 安装完成后，打开任意一篇长文网页，您将看到右下角的蓝色悬浮球。
2. 点击打开 AI 面板，并点击右上角的 **⚙️ 齿轮图标** 进入设置。
3. 填入您的 `API Key` 并检查/修改 `API Base URL`。
4. 点击 **[获取列表]** 拉取可用的模型，挑选您最爱的对话模型（如 `gpt-4o` 或 `deepseek-chat`）。
5. 点击底部 **[保存设置]**！现在，开始与网页对话吧！
