# AI 网页大模型助手 (AI Web Page Assistant)

🔥 让你直接在长文中跟网页“对话”的终极摸鱼/学习神器！

借助大模型的能力，这个油猴脚本把你的浏览器变成了带 AI 外挂的全能阅读器。看到几万字的长篇大论不想看？直接点开右下角让 AI 帮你做总结。想知道文章细节？选中文本片段直接让它解释！

## 🌟 核心亮点

- **懂网页的 AI**：不再傻乎乎地复制粘贴。脚本内置了 `Readability.js`，能像人一样精准剥离网页正文，自动忽略广告、侧边栏和底部的废话，把最纯净的内容喂给 AI。不仅精准度提高，还能帮你省下一大笔 Token 费用。
- **极致清爽体验**：
  - **绝不碍事**：右下角的悬浮按钮平时安安静静，鼠标不放上去 3 秒后就会半透明躲在屏幕边缘。按住还能随便拖，想放哪就放哪。
  - **优雅面板**：一键呼出（或者用快捷键 `Ctrl+J` 打得更爽）。电脑上是从右边滑出来的抽屉，手机上则是底部抽起，完全不破坏网页原本的布局。
- **自定义，玩出花**：
  - **不绑死任何平台**：只要兼容 OpenAI 格式的接口（DeepSeek, 通义千问, 阿里百炼代理等）统统能接。填上你的 `API Base URL` 和 `API Key`，完全由你做主。
  - **支持打字机流式输出**，看字一个个蹦出来的快感拉满。
- **百变模型管家**：
  - 自带 6 个超实用的预设 prompt（📝 通用总结、💻 技术文档、📰 新闻文章、📚 教程指南、🔬 学术论文、🎁 产品介绍）。
  - 当然，你也可以在设置里按照你的喜好去自己捏“专属助手”。
- **比你还懂你想问啥**：自动根据你跟这段网页的聊天上下文，在底部生成 3 个犀利的【追问气泡】。不用打字，点一下就自动追问。

## 安装指南

1. **第一步**：你得有一个装好了 [Tampermonkey (油猴插件)](https://www.tampermonkey.net/) 的浏览器。
2. **第二步**：[一键点击安装脚本](https://raw.githubusercontent.com/lilynas/AI-Web-Page-Assistant/main/ai_web_assistant.user.js)，Tampermonkey 会自动识别，点击安装即可！

## 简单上手教程

1. 随便找一篇干货文章点进去，注意看右下角，那颗蓝色的“小脑袋”在等你。
2. 点击它展开面板，然后点一下最上面的 **⚙️ 齿轮图标** 进入属于你的设置专区。
3. 填好你的 `API Key`（如果你是用第三方代理，记得改一下 `API URL`）。
4. 点击一下 **[获取列表]**，选一个顺眼的模型（比如今天很能打的模型）。
5. 最后点 **[保存设置]** 搞定！

现在，返回聊天界面，点击那个闪闪发光的 **"✨ 总结全文"** 开始感受魔法吧！
